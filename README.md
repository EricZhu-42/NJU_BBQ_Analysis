# NJU_BBQ_Analysis

> 南京大学 2019春季学期 人工智能程序设计 课程设计作业

以下为目录内文件/文件夹的简要介绍

文件/文件夹名称 | 简介
:--: | :--: 
data/ | 导出的部分数据
depedencies/ | 创建词云时依赖的第三方文件
models/ | 保存的词向量矩阵，词向量转换器等模型
output/ | 导出的分析结果图片等
*.py | 项目涉及到的python脚本代码

另：本项目中部分代码与功能的实现请参考：[QZone_Spider](<https://github.com/EricZhu-42/QZone_Spider>).
